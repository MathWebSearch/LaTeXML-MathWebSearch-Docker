# LaTeXML-MathWebSearch-Docker

A Docker Image for the LaTeXML part of MathWebSearch. 

It is published on DockerHub as the automated build [mathwebsearch/latexml-mws](https://hub.docker.com/r/mathwebsearch/latexml-mws) and can be run as:

```
    docker run -p 8080:8080 mathwebsearch/latexml-mws
```
